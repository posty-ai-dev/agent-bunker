# Getting Started

Welcome to Agent Bunker! This is your workspace where you can store agent memory, configuration, and notes.

## Quick Tips

- Use the **Editor** tab to edit files in this workspace
- Use the **Cron & Jobs** tab to manage scheduled tasks
- Use the **Activity** tab to track agent operations
- Use the **Agents Playground** for a live overview of all agent activity

## Customization

Edit the `.env` file to point to your own data directories and configure integrations.
