# Example Skill

This is a sample skill that demonstrates the skills registry feature.

## Usage

Skills are discovered automatically by scanning directories listed in `SKILL_DIRS`.
Each skill should have a `SKILL.md` file with YAML frontmatter containing `name`, `description`, and `emoji` fields.
